# 视频提示词 Skill

用于最终 `video` 节点。它只处理 video prompt、时长、画幅、制作路径和引用关系。

## 读取素材

写视频 prompt 前读取或确认：

- 视频总时长、画幅、视频类型、整体视觉风格和目标节奏。
- 剧情段落、角色关系、动作目标、情绪变化、对白需求和结尾画面。
- 文生视频所需的主体设定、场景设定、光线、天气、空间关系和声音目标。
- 图生视频所需的分镜图、首帧图、尾帧图、人物图、场景图和其他参考图的用途。
- 分镜图的真实宫格布局、阅读顺序、每格主体、构图、动作瞬间、景别和视线方向。
- 当前视频模型支持的参考图数量、时长字段和画幅字段。

分镜图、首帧或尾帧有真实图片输出时，先用视觉能力查看图片。根据图中实际行列判断布局，读取顺序为从左到右、从上到下。视觉信息缺失时，用分镜 image 节点 prompt 和上游剧情作为有限依据。

## 通用写法

prompt 写成连续视频描述。先写视频类型、主体、场景、整体风格和参考图片用途；再按时间顺序写每个镜头或分镜格的动作、镜头变化、景别、构图、转场、节奏、物理反馈和声音。

视频模型更容易执行具体、可见、连续的画面描述。每个时间段按三层组织：

- 镜头层：景别、机位、镜头运动和速度。
- 主体层：一个核心动作、表情或姿态变化、动作速度和动作结果。
- 画面层：光线方向、环境运动、自然元素、道具反馈和空间连续性。

常用景别：大远景、远景、全景、中景、中近景、近景、特写、大特写。

常用镜头运动：静止、缓推、拉远、环绕、跟拍、横移、上摇、下摇、手持、俯冲。

常用光线：黄金时刻、蓝色时刻、柔和窗光、霓虹光、月光、烛光、逆光、侧光、顶光。

动作描述使用具体动词和速度副词，例如“缓缓转头”“轻轻抬手”“突然后撤”“渐渐握紧”“微微低头”“快速越过门槛”。每个时间段安排一个核心动作，并允许一个自然动作结果。

## 文生视频格式

纯文生视频、没有可用参考图或用户要求直接根据文字创意生成视频时，使用这个格式。写入 `node.prompt` 时替换花括号，不保留模板占位符。

```text
生成一个{duration}、{aspect_ratio}、{shot_count}个镜头的{video_type}。

视频类型：
{电影短片 / 黑暗奇幻预告片 / 真实纪录片 / 一镜到底 / 短剧 / 广告片等具体类型。}

主体设定：
{主体是谁，外貌、年龄感、服装、道具、姿态、情绪、当前状态和关键识别点。纯文生视频需要写清楚可见特征，保证模型能稳定保持主体。}

场景设定：
{地点、时间、天气、空间结构、主体和环境的位置关系、主要光源、背景活动和可见动态元素。}

整体风格：
{色调、光影、节奏、质感、摄影风格、画面情绪、真实感或动画质感。}

镜头一 —— {镜头名}，{开始}-{结束}秒，{景别}，{机位/运镜}。
{主体在哪里，动作如何开始，主体做什么，环境如何运动，动作造成什么物理结果，镜头如何配合动作。}

转场：
{用动作方向、遮挡、光线、焦点、声音、视线或空间位置自然承接到下一个镜头。}

镜头二 —— {镜头名}，{开始}-{结束}秒，{景别}，{机位/运镜}。
{承接上一个镜头继续写动作发展、物理反馈、环境变化和镜头运动。}

声音设计：
{环境声、动作声、道具声、氛围声、音乐情绪、对白/旁白/字幕要求。}

稳定性要求：
全程保持同一主体、同一服装、同一道具、同一场景逻辑、同一光源方向和统一色调。动作缓慢连贯，镜头稳定，人体结构正常，面部清晰，布料、头发、烟雾、风雪、水面、火花、粒子和光效等运动自然真实。
```

15 秒短片通常使用 3-5 个镜头：建立镜头 2-4 秒，动作发展 4-8 秒，收束镜头 3-5 秒。一镜到底视频使用一个镜头段落，重点写主体运动路线、镜头运动路线、空间变化、环境动态和结尾停留画面。

## 分镜图生视频格式

根据已完成的分镜图、宫格分镜、首尾帧、参考图和剧情，为 `video` 节点写最终提示词时，使用这个格式。分镜图只作为镜头构图和动作节奏参考，最终画面是连续视频。

```text
图片引用：
图片1：{人物/场景/首帧/尾帧/分镜图/风格图}参考，{说明它锁定的主体、场景、构图或动作信息}。
图片2：{用途}参考，{说明它锁定的信息}。

整体设置：
{duration}秒，{aspect_ratio}，{video_type}。视觉风格为{style}，画面保持{色调、光线、质感、节奏和摄影风格}。参考分镜图的真实宫格顺序组织镜头，成片是连续视频画面。

0-3秒 | 第1排第1格 | {画面概述} -> {动作变化} | {镜头运动}
{景别}，{机位}，镜头{缓推/拉远/环绕/跟拍/横移/上摇/下摇/手持/俯冲/静止}。{主体从什么状态开始，以什么速度做一个核心动作，表情、姿态、道具和服装如何变化。}{光线方向、环境细节、风、雾、水面、火花、布料或头发等自然元素如何运动。}

3-7秒 | 第1排第2格 | {画面概述} -> {动作变化} | {镜头运动}
{按分镜下一格继续写动作承接、运动方向、镜头变化、主体情绪和环境反馈。}

声音设计：
{环境声、动作声、道具声、材质声、氛围声、音乐情绪、对白或旁白。}

一致性要求：
全程保持同一主体、同一服装、同一道具、同一场景逻辑、同一光源方向和统一色调。动作缓慢连贯，镜头稳定，人物结构自然，面部清晰，布料、头发、烟雾、风雪、水面、火花和光效运动真实。
```

图片引用只写用户可理解的图片编号和用途。正文中使用人物名、场景名或物体名指代参考内容，例如“小明”“竹林”“青铜剑”，让正文保持自然连贯。

## 时间分配

每个分镜格对应一个时间段。先根据分镜图实际格数和剧情密度分配时长，再写每段提示词。动作起势、空间移动、打斗、追逐、变形、爆炸、落地、转身、发现和情绪转折可以分配更长时间。所有时间段的总和等于 `duration_seconds`。

15 秒四宫格常用节奏：`0-3秒` 建立，`3-7秒` 起势和推进，`7-12秒` 转折或高潮，`12-15秒` 结果和停留。六宫格或九宫格按动作密度重新分配。

首帧或尾帧存在时，首段开头对齐首帧的主体位置、构图和光线，末段结尾对齐尾帧的主体姿态、空间关系和情绪落点。分镜图已经提供静态构图时，提示词重点写运动过程和动作变化。

对白或旁白存在时，把台词写入对应时间段，注明说话人、语气和动作状态。

## 视频类型写法

电影短片强调戏剧动作、光影、景别变化和结尾画面。

黑暗奇幻预告片强调低照度、雾气、符文、火光、慢节奏推进、压迫感和史诗氛围。

真实纪录片强调自然光、真实环境声、手持或稳定跟拍、细节观察和生活化动作。

一镜到底强调连续空间、连续动作、镜头不切、运动路径清晰和自然转身/遮挡/焦点变化。

短剧强调人物关系、表情、对白节奏、近景和可读的情绪变化。

广告片强调产品或主体亮点、材质细节、干净构图、节奏递进和记忆点收束。

## Standalone 节点推荐字段

以下字段只用于直接创建或更新 `video` 节点。Workflow Build Mode 只把时长、用途、制作路径和引用等业务字段写进 spec；模型、画幅、分辨率、宽高、画质和 fps 由前端运行配置提供。

- `fields.purpose`: `final_video` 或当前段落用途
- `fields.stage`: `video`
- `fields.prompt_source`: `skill:video_prompt`
- `fields.prompt_status`: `final_prompt`
- `fields.production_path`: `text_to_video` 或 `image_to_video`
- `fields.duration_seconds`: 用户指定或当前段落时长
- `fields.aspect_ratio`: 用户指定或已确认画幅
- `fields.references`: 剧本/段落故事用 `context`，视觉参考图用 `visual_reference`

## 检查点

- prompt 能直接交给视频模型执行。
- 文生视频从视频类型、主体设定、场景设定和整体风格开始。
- 分镜图生视频根据真实宫格行列和阅读顺序组织时间段。
- 每个时间段都有时间、景别、机位、运镜、主体动作、物理反馈和环境动态。
- 时长总和等于 `duration_seconds`，画幅和 references 与节点字段一致。
- 成品提示词只保留视频模型需要执行的画面、声音和一致性描述。
