# OpenReel Studio 安装器

[English](./README.md) · 简体中文

这是官方 npm 安装器，用于从 GitHub Releases 下载最新桌面安装包，或引导 Linux 服务器部署。

下载当前桌面平台的安装包：

```bash
npx openreel-studio-installer
```

引导 Linux 服务器部署：

```bash
npx openreel-studio-installer server
```

常用选项：

```bash
npx openreel-studio-installer --dir ./downloads
npx openreel-studio-installer --no-open
npx openreel-studio-installer --print-url
```
